== Description ==

Members Management Pro - это мощный плагин для WordPress, предназначенный для управления базой участников и экспертов вашего сообщества или ассоциации.

= Основные возможности =

* **Таксономии для фильтрации** - Типы участников, Роли в ассоциации, Локации
* **Продвинутая страница участников** - с фильтрами, поиском и красивыми карточками
* **Персональные страницы** - детальная информация о каждом участнике
* **Встроенный импорт** - прямо из админки WordPress
* **Адаптивный дизайн** - отлично выглядит на всех устройствах
* **AJAX фильтрация** - мгновенная фильтрация без перезагрузки страницы
* **Раздел материалов** - публикации, видео и методики для каждого участника

= Использование =

После установки используйте шорткод на любой странице:

`[members_directory]`

Параметры шорткода:
* `show_filters="yes"` - показывать фильтры (yes/no)
* `columns="3"` - количество колонок (2/3/4)
* `show_search="yes"` - показывать поиск (yes/no)

Пример:
`[members_directory show_filters="yes" columns="3" show_search="yes"]`

= Шаблон персональной страницы =

Для отображения персональных страниц участников скопируйте файл `single-members.php` из папки плагина в папку вашей темы.

== Installation ==

1. Загрузите папку `members-management-pro` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Скопируйте `single-members.php` в папку вашей темы (опционально)
4. Перейдите в раздел "Участники сообщества" для добавления участников
5. Используйте встроенный импортер для массовой загрузки данных из CSV
6. Создайте страницу и добавьте шорткод `[members_directory]`

== Frequently Asked Questions ==

= Как импортировать участников из CSV? =

1. В админке перейдите в "Участники сообщества → Импорт из CSV"
2. Загрузите файл CSV (пример включен в плагин)
3. Нажмите "Импортировать"

= Не отображаются участники на странице =

1. Убедитесь, что плагин активирован
2. Сбросьте постоянные ссылки: Настройки → Постоянные ссылки → Сохранить
3. Проверьте, что на странице есть шорткод `[members_directory]`

= Как настроить внешний вид? =

Плагин использует встроенные стили, которые можно переопределить в файле style.css вашей темы. Все CSS классы задокументированы в файле DOCUMENTATION.md

= Как добавить поля для участников? =

Используйте хуки WordPress:
`add_filter('members_meta_fields', 'your_function');`

Подробнее в DOCUMENTATION.md

== Technical Details ==

**Типы данных:**
* Custom Post Type: `members`
* Taxonomies: `member_type`, `member_role`, `member_location`
* Meta Fields: position, company, email, phone, bio, и др.

**Требования:**
* WordPress 5.0 или выше
* PHP 7.2 или выше
* jQuery (включено в WordPress)

**Поддержка:**
* Мультиязычность: готов к переводу
* REST API: полная поддержка
* SEO: Schema.org разметка
* Accessibility: WCAG совместимость
