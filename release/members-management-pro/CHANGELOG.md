# Changelog

Все значительные изменения в этом проекте будут задокументированы в этом файле.

## [2.1] - 2025-11-18

### Добавлено
- Кнопка "Отправить сообщение" в блоке контактов на странице профиля
- Полнофункциональные табы в разделе Материалы (Публикации, Видео, Методики)
- Интерактивные элементы материалов - кликабельные карточки и кнопки "подробнее"

### Улучшено
- **Архивная страница:**
  - Стили кнопок фильтров: неактивные кнопки теперь серые (#999), активные - синие (#007cba) с белым текстом
  - Уменьшена толщина бордюра кнопок фильтров с 2px до 1px для более современного вида
  - Добавлен white-space: nowrap для предотвращения переноса текста на кнопках

- **Выпадающие списки:**
  - Исправлена позиция стрелки в фильтрах "Город" и "Роль в ассоциации"
  - Стрелка теперь позиционирована с отступом 12px от правого края
  - Увеличен правый padding до 40px для корректного отображения

- **Карточки участников:**
  - Убрано автоматическое сокращение текста в полях "Должность" и "Компания"
  - Добавлены свойства word-wrap и overflow-wrap для многострочного отображения
  - Выровнены баджи с ролями по левому краю карточки (justify-content: flex-start)

- **Цветовая схема:**
  - Добавлена защита от отображения малинового цвета на кнопках "Показать еще" при активном состоянии
  - Принудительная установка цвета #007cba для всех кнопок кроме фильтров

- **Страница профиля:**
  - Выровнен font-weight во всех текстовых блоках (установлен единый font-weight: 400)
  - Убран italic стиль из блока биографии для лучшей читаемости
  - Улучшена визуальная консистентность между разделами "О себе" и "Ожидания от сотрудничества"

### Исправлено
- Позиционирование стрелок в селектах фильтров
- Проблема с переполнением текста в карточках участников
- Некорректное выравнивание ролевых бейджей
- Несоответствие стилей кнопок при наведении и активном состоянии
- Визуальное несоответствие между текстовыми блоками профиля

## [2.0] - 2024

### Добавлено
- Система таксономий для фильтрации (member_type, member_role, member_location)
- AJAX фильтрация и поиск участников
- Персональные страницы участников с детальной информацией
- Встроенный импортер для загрузки данных из CSV
- Метабоксы для всех полей участников
- Адаптивный дизайн для всех устройств
- Шорткод [members_directory] с параметрами
- Раздел материалов на странице профиля

### Основной функционал
- Custom Post Type "members" с полной поддержкой
- 3 таксономии: тип участника, роль в ассоциации, локация
- Метаполя: должность, компания, email, телефон, LinkedIn, вебсайт, биография
- Гибкая система фильтрации
- Адаптивная сетка участников (2/3/4 колонки)
- Поддержка импорта из CSV
- Административный интерфейс для управления участниками
